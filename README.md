## 🛠️ Stack Tecnológico del Proyecto

Aunque soy un ingeniero especializado en Backend, este frontend ha sido diseñado siguiendo principios de simplicidad y rendimiento.

* **HTML5:** Estructura semántica enfocada en accesibilidad.
* **CSS3 (Variables/Grid/Flexbox):** Diseño minimalista, totalmente responsivo y "Mobile-First".
* **Vanilla JS (ES6+):** Interactividad ligera (como el "Dark Mode Toggle"), sin frameworks pesados para garantizar tiempos de carga ultrarrápidos.
* **Iconos:** FontAwesome.
* **Despliegue:** Aproveché las bondades de GitHub Pages.

## 🔧 Ejecución Local

Si deseas clonar este repositorio y ejecutarlo localmente, no necesitas ninguna dependencia compleja.

1.  Clona el repositorio:
    ```bash
    git clone https://github.com/jmsierra26/jmsierra26.github.io.git
    ```
2.  Navega a la carpeta:
    ```bash
    cd jmsierra26.github.io
    ```
3.  Abre el archivo `/docs/index.html` directamente en tu navegador favorito.
    * *Opcional para desarrollo:* Usa una extensión como "Live Server" en VS Code para recarga automática.
